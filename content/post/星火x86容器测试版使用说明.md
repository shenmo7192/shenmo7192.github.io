#### 仅供测试

支持：aarch64,理论上支持其他的架构，未测试

### 效果：可在非X86上运行X86应用

#### 步骤

1. `sudo apt install qemu-user-static binfmt-support sshpass  systemd-nspawn -y `

安装qemu，binfmt和nspawn依赖

2. 在这里下载容器包 ，下载后解压到主目录（$HOME），以后的操作假设你已经解压到了主目录。如果你有动手能力，可以放到其他目录，接下来的教程按需修改
3. `cd ~ && sudo systemd-nspawn -bD deepin`，等待容器环境启动结束弹出登陆
4. 保持此终端不关闭，另开终端，输入 `sshpass -p 986532147 ssh -X spark@localhost  `
5. 在x86环境进行一系列操作吧
